# Game Music Player

Automated music player for the game "Where Winds Meet".

## Features
- User-friendly GUI for playing and converting music
- PDF and MIDI file support for automatic note transcription
- Jianpu (numbered) notation and keyboard mapping
- Adjustable note delay, tempo, and playing modes (Guqin, Fast, Custom)
- Semitone support (Shift/Ctrl for #/b notes)
- Real-time note validation and status log

## Quick Start
1. Install Python and dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Run the GUI:
   ```
   python music_player_gui.py
   ```
3. Use the interface to upload PDF or MIDI files, paste notes, and play them in-game.

## Note Mapping
- High: Q W E R T Y U
- Medium: A S D F G H J
- Low: Z X C V B N M
- Semitones: Hold Shift for #, Ctrl for b

## Usage
- Use the GUI to upload, convert, and play music.
- Adjust settings for delay, tempo, and mode as needed.
- Focus the game window before playback.
